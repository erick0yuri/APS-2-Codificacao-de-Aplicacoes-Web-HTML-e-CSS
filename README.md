# APS 2 - Codificação de Aplicações Web HTML e CSS

Publicação das atividades desenvolvidas ao longo da disciplina de Codificação de Aplicações Web, utilizando GitHub Pages.

## Sobre

Este repositório reúne os exercícios práticos de HTML e CSS feitos em sala de aula, além do projeto final da disciplina.

## Estrutura

- **cap3/** - Exercícios de HTML básico (cabeçalhos, imagens, links e listas)
- **cap4/** - Float e Design Responsivo
- **cap5/** - Formulário, Layout e Tabela
- **aparecida-nutricao/** - Projeto final: sistema de cadastro de pacientes com cálculo de IMC

## Acesso

O site está publicado via GitHub Pages:

https://erick0yuri.github.io/APS-2-Codificacao-de-Aplicacoes-Web-HTML-e-CSS/

## Tecnologias

- HTML5
- CSS3
- JavaScript (projeto final)

A página inicial (`index.html`) reúne o acesso a todas as atividades, organizadas por capítulo.
